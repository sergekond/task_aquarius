# Программа для подсчёта количества единичных битов в любом файле.
## Работает с любыми типами данных: текст, бинарные файлы, изображения, видео, архивы и т.д.

### Возможности
* Подсчёт количества битов, равных 1, во всём файле.
* Поддержка любых файлов (чтение в бинарном режиме).
* Алгоритм подсчёта через таблицу popcount[256].
* Минимальные зависимости — используется только стандартная библиотека C.
* Сборка через bash или cmake.
* Корректная обработка ошибок (нет файла, нет доступа, пустой файл и т.д.).

## Тестирование
### Через CMake
Необходимо в ```launch.json``` в поле ```"args"``` указать путь к файлу, в котором необходимо подсчитать количество единичных битов.
После чего собрать проект, можно в bash 
```
cd build
cmake ..
cmake --build
```

### Через bash
Собрать проект 
```
mkdir build
cd build
cmake ..
cmake --build
```

Перейти в workspace директорию ```cd ../```

вызвать ```main.exe``` через ```./main <название вашего файла>```

При корректной работе программа выводит: ```file <название вашего файла> contains <количество 1> ones.```
При неисправности программа выводит: ```Error opening file <название вашего файла>: <Вид ошибки>\n```
